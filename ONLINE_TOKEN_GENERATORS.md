# 在线安全令牌生成器

## 推荐的在线生成器

### 1. Random.org
- 网址：https://www.random.org/strings/
- 设置：8个字符串，每个8个字符，使用十六进制字符
- 生成：64个字符的随机令牌

### 2. 密码生成器
- 网址：https://passwordgenerator.net/
- 设置：64个字符，只使用0-9, a-f
- 生成：十六进制随机令牌

### 3. Bitwarden 密码生成器
- 网址：https://bitwarden.com/password-generator/
- 设置：64个字符，十六进制字符集

## 使用步骤

1. 访问上述任一网站
2. 设置参数生成64个字符的随机令牌
3. 复制生成的令牌
4. 在Railway控制台中添加环境变量：
   ```
   GATEWAY_TOKEN=your_generated_token_here
   ```

## 注意事项

⚠️ **安全警告**：
- 不要使用简单的密码或可预测的字符串
- 确保令牌长度至少32个字符
- 不要在代码中硬编码令牌
- 定期更换令牌（建议每30天）

## 令牌要求

- **长度**：至少32个字符（推荐64个）
- **字符集**：0-9, a-f（十六进制）或更广泛的字符集
- **唯一性**：确保令牌是唯一的
- **安全性**：使用密码学安全的随机数生成器
